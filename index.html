Puke

Puke es un manejador de archivos compresos similar a 7zip, con muchas funcionalidades, con la ventaja de estar escrito en .Net y con librerias de codigo abierto.

Download this project as a .zip file Download this project as a tar.gz file

Introducción

La idea de crear una aplicacion que comprimera/descomprimiera y empaquetara nacio luego de abandorar una faena de adiccion hacia un juego cúyo nombre no mencionare, retomé un proyecto que anteriormente estaba creando que se encargaria de descomprimir archivos contenedores de scripts para un juego.
